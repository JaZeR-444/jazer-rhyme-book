# Scripts Catalog

This directory contains various scripts for maintaining and building the JaZeR Rhyme Book.

## Subdirectories

- **build/**: Scripts for generating indexes, validating data, and preparing web assets.
- **data/**: Utilities for adding and expanding the dictionary and knowledge base.
- **maintenance/**: Tools for cleaning up, fixing, and migrating data.
- **utils/**: General-purpose helper scripts.
