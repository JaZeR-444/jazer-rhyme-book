# WORD: Asm

## Meaning (plain):
1. (Computing) Abbreviation for Assembly Language, a low-level programming language closely related to machine code. 2. (Military) Air-to-surface missile or Anti-ship missile. 3. (Theatre) Assistant Stage Manager.

## Rap meaning (how I'd say it):
Metaphor for working at a "low level" (the foundational, raw code of the game) or for a precision strike (missile). Represents technical mastery and the ability to trigger "explosive" results with precise commands.
